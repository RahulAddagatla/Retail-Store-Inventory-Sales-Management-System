
                                               Retail Store Inventory
                  
This project implements a fully normalized relational database system designed to track, manage, and analyze
operational data for a multi-category retail storefront. It models standard enterprise workflows encompassing
customer registration, dynamic inventory tracking, sales transactions, and multi-item order fulfillment
schemas.
The core objective is to showcase programmatic database design methodologies and translate unstructured
transaction logs into actionable analytical insights using advanced SQL mechanisms like multi-table joins,
conditional aggregations, subqueries, and windowing patterns.
